• AimShield bij drive-by's staat volledig uit. Daardoor kun je silent aim met een unlimited FOV gebruiken. Anders zouden er extreem veel false detections zijn. Je hoeft niet eens direct op personen te richten, dus je kunt echt met een unlimited FOV spelen.

• AimShield bij het schieten op mensen in voertuigen staat ook uit. Bijvoorbeeld: als jij uitstapt en schiet op mensen die in een auto, heli, boot of motor zitten, word je nooit gedetecteerd. Dit staat uit omdat het anders veel false detections zou veroorzaken. Ook hier hoef je niet precies op personen te richten, dus je kunt een unlimited FOV gebruiken.

• Voet-tegen-voet shootouts (bijvoorbeeld wanneer beide partijen uitgestapt zijn):
Je kunt hier nog steeds silent aim gebruiken, zolang je binnen een halve seconde na het legit richten schiet.  
Dus eerst legit op iemand aimen en daarna binnen een halve seconde de kogel lossen.

Deze methode werkt het beste voor long-range fights.

• AimShield bij shotguns staat volledig uit. Daardoor kun je met shotguns ook een unlimited FOV gebruiken, omdat de shells anders voor veel false detections zouden zorgen.

De enige manier waarop je gedetecteerd kunt worden, is bij voet-tegen-voet fights wanneer je niet daadwerkelijk op iemand hebt gericht, maar hem toch raakt met silent aim.
